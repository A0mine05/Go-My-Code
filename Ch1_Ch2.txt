#Checkpoint 1 : Comparaison

Data Analyst :	
Avec les outils d'analyse, le data analyst doit récupérer, organiser et  interpreter, les données statistiques afin que le client puisse prendre des décisions stratégiques nécessaires pour le developpement de son business.	

Data Scientist :
En plus des responsabilités d'un data analyst, le data scientist dispose de connaissances approfondies dans le domaine du machine learning et du deep learning. Ce qui lui permet de garder la main sur les architectures, comprendre la programmation et deployer des modeles.
	
Data Engineer :
Le data engineer est chargé de préparer les données afin que le data analyst et scientist ne les utilisent pour leur analyse. Il traite les données venant de divers sources (optimiser, intégrer, gérer les modeles opérationnels...).
